# 3D Книга

![Preview](https://cdn.poehali.dev/templates/book-rendering-ru/preview-ru.jpg)

Красивый шаблон для книжного магазина с эффектной 3D-книгой, которая вращается прямо на экране. Идеально подходит для презентации книг, создания магазина или промо-страницы издательства.

На странице показывается книга в объёмном виде — её можно рассмотреть со всех сторон. Рядом отображается вся информация: название, автор, описание, рейтинг и кнопки для добавления в корзину или избранное. Переключаться между книгами можно стрелками — при этом книга красиво переворачивается, а фон плавно меняет цвет под стиль обложки.

В шаблоне уже настроено 4 книги с разными обложками и цветовыми темами. Вы можете легко заменить их на свои — просто добавьте изображения обложек и отредактируйте данные в файле. Дизайн адаптивный: на телефоне книга показывается сверху, а описание снизу.

Шаблон отлично подойдёт для книжных магазинов, издательств, авторов которые хотят красиво представить свою книгу, или для любого проекта, где нужен вау-эффект с 3D-графикой.
